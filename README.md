### Функционал
POST - запрос на /api/counter возвращает строковое представление подсчета вхождений символов в строку запроса

### Запуск
- клонировать репозиторий себе на пк
- запуск в ide
- сборка Maven
- swagger - файл: countApi.yaml

### Тестирование
Предусмотрены postam-тесты в папке "Postman".
